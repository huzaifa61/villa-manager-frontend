# Villa Manager Frontend - React Native Mobile App

Complete mobile application for Villa Manager Pro.

## Setup

```bash
cd villa-manager-frontend
npm install
npm start
npm run android  # or npm run ios
```

## Features

- Complete authentication flow
- Dashboard with financial metrics
- Apartment management
- Payment & expense tracking
- Service requests
- Reports & export
- User control
